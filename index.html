สวัสดีค่ะ ฉันชื่อ ธิดารัตน์ ตาราษี ชื่อเล่น มายมิ้น อายุ19ปี
เป็นนักศึกษาคณะวิศวกรรมคอมพิวเตอร์และหุ่นยนต์
กำลังศึกษาอยู่ที่มหาวิทยาลัยกรุงเทพ ฉันมีความชอบในด้านการออกแบบเว็บไซต์ UI/UX และการพัฒนาเว็บไซต์

ฉันมีความชื่นชอบการเรียนวิชาคอมพิวเตอร์ตั้งแต่สมัยประถมศึกษาและ
ได้มีโอกาสเขียนโค้ดครั้งแรกเมื่อตอนมัธยมศึกษาปีที่1 ซึ่งฉันได้เรียนรู้
และพัฒนาทักษะการเขียนโค้ดมาตลอดจนถึงปัจจุบัน โดยตลอดระยะ
เวลาที่ผ่านมาฉันได้ศึกษาภาษาอื่นๆนอกเหนือจากที่ได้เรียนมาในรั้ว
โรงเรียนไม่ว่าจะเป็น PHP JavaScript python C แต่ภาษาที่ฉันถนัด
มากที่สุดจะเป็น html และ css
